Summary: Spotify Music Popularity

We utilized spotify track listings via Kaggle to attempt to predict and determine if the song will be popular or not using the danceability, loudness,speechiness, acousticness and various other musical attributes.



Final Project: Structure and Parameters

* Regression Analysis to make a prediction on the most popular songs or just a song in general
* Regression Model for the Machine Learning aspect of the project
* Tableau for visualizations
* Amazon AWS and SQL for databases
* Heroku as the Host Application
